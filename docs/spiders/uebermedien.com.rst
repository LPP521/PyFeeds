uebermedien.com
---------------
Newest articles from Übermedien_.

Configuration
~~~~~~~~~~~~~
Add ``uebermedien.com`` to the list of spiders:

.. code-block:: ini

   # List of spiders to run by default, one per line.
   spiders =
     uebermedien.com

.. _Übermedien: http://www.uebermedien.de
